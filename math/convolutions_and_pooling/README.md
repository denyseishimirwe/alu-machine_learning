# Convolutions and Pooling

This project covers valid/same/padded/strided convolutions on grayscale and
multi-channel images, multi-kernel convolution, and max/average pooling.

## Requirements

- Ubuntu 16.04 LTS
- Python 3.5
- NumPy
- pycodestyle 2.5

Only the allowed number of `for` loops may be used in each function.

## Files

| File | Description |
| --- | --- |
| `0-convolve_grayscale_valid.py` | Valid grayscale convolution. |
| `1-convolve_grayscale_same.py` | Same grayscale convolution. |
| `2-convolve_grayscale_padding.py` | Grayscale convolution with custom padding. |
| `3-convolve_grayscale.py` | Strided grayscale convolution. |
| `4-convolve_channels.py` | Convolution with channels. |
| `5-convolve.py` | Convolution with multiple kernels. |
| `6-pool.py` | Max / average pooling. |
