# Probability

This project covers common probability distributions used in machine
learning: Poisson, Exponential, Normal, and Binomial.

## Requirements

- Ubuntu 16.04 LTS
- Python 3.5
- NumPy (for test scripts only)
- pycodestyle 2.5

No external modules may be imported in the distribution classes.

## Files

| File | Description |
| --- | --- |
| `poisson.py` | Poisson distribution (PMF, CDF). |
| `exponential.py` | Exponential distribution (PDF, CDF). |
| `normal.py` | Normal distribution (z-score, PDF, CDF). |
| `binomial.py` | Binomial distribution (PMF, CDF). |
