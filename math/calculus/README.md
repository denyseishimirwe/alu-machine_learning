# Calculus

This project covers foundational calculus concepts for machine learning,
including sigma and product notation, derivatives, partial derivatives,
integrals, and definite integrals.

## Requirements

- Ubuntu 16.04 LTS
- Python 3.5
- pycodestyle 2.5

All scripts start with `#!/usr/bin/env python3`, are executable, end
with a newline, and follow `pycodestyle`. Modules and functions are
documented.
