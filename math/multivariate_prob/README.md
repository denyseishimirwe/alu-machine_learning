# Multivariate Probability

This project covers mean, covariance, correlation, and the multivariate
normal distribution.

## Requirements

- Ubuntu 16.04 LTS
- Python 3.5
- NumPy
- pycodestyle 2.5

## Files

| File | Description |
| --- | --- |
| `0-mean_cov.py` | Mean and covariance of a data set. |
| `1-correlation.py` | Correlation matrix from covariance. |
| `multinormal.py` | Multivariate Normal distribution (PDF). |
