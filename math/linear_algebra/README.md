# Linear Algebra

This project covers the basics of linear algebra for machine learning,
including vectors, matrices, transposition, element-wise operations,
concatenation, dot products, matrix multiplication, and core `numpy`
operations such as shape inspection, transposition, broadcasting, and
matrix multiplication.

## Requirements

- Ubuntu 16.04 LTS
- Python 3.5
- numpy 1.15
- pycodestyle 2.5

## Files

| File | Description |
| --- | --- |
| `0-slice_me_up.py` | Slice a list into three sub-arrays. |
| `1-trim_me_down.py` | Extract the middle two columns of a 2D list. |
| `2-size_me_please.py` | Return the shape of a (nested) list. |
| `3-flip_me_over.py` | Transpose a 2D list. |
| `4-line_up.py` | Element-wise add two arrays. |
| `5-across_the_planes.py` | Element-wise add two 2D matrices. |
| `6-howdy_partner.py` | Concatenate two arrays. |
| `7-gettin_cozy.py` | Concatenate two 2D matrices along an axis. |
| `8-ridin_bareback.py` | Multiply two 2D matrices. |
| `9-let_the_butcher_slice_it.py` | Numpy slicing examples. |
| `10-ill_use_my_scale.py` | Return the shape of a `numpy.ndarray`. |
| `11-the_western_exchange.py` | Transpose a `numpy.ndarray`. |
| `12-bracin_the_elements.py` | Element-wise add/sub/mul/div with numpy. |
| `13-cats_got_your_tongue.py` | Concatenate two `numpy.ndarray`s. |
| `14-saddle_up.py` | Multiply two `numpy.ndarray`s. |
