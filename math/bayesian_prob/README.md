# Bayesian Probability

This project covers Bayesian inference for a binomial likelihood:
likelihood, intersection with a prior, marginal probability, and posterior.

## Requirements

- Ubuntu 16.04 LTS
- Python 3.5
- NumPy
- pycodestyle 2.5

## Files

| File | Description |
| --- | --- |
| `0-likelihood.py` | Binomial likelihood over hypothetical probabilities. |
| `1-intersection.py` | Likelihood × prior (intersection). |
| `2-marginal.py` | Marginal probability of the data. |
| `3-posterior.py` | Posterior probabilities given the data. |
